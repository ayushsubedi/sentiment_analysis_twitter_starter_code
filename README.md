# Twitter Sentiment Analysis

### Getting data from twitter
1. Using Tweepy (uses official twitter API)
- easy
- several limitations
- check https://developer.twitter.com/en/docs/rate-limits

2. Using Twint (unofficial)
- a little difficult
- less limitation
- amazing for large volume
